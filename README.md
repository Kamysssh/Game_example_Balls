# 🎨 Игра про шарики

Интерактивная игра, где вы можете управлять разноцветными шариками, которые движутся по экрану и смешивают цвета при касании.

## 📋 Описание

Игра состоит из двух основных компонентов:
- **logic.py** - чистая игровая логика без графики
- **game.py** - графический интерфейс на pygame

### Возможности игры:

✨ **Анимированные шарики** - плавное движение с отскоком от стен  
🎨 **Смешивание цветов** - при касании шарики смешивают цвета  
🖱️ **Управление мышью** - всасывание и перетаскивание шариков  
📦 **Интерактивный инвентарь** - перетаскивайте шарики мышью обратно на поле  
🗑️ **Зона удаления** - перетащите шарик в красную зону для удаления  
💎 **3D графика** - объёмные шарики с градиентами, бликами и тенями  

## 🚀 Установка и запуск

### 1. Установите зависимости:

```bash
pip install -r requirements.txt
```

### 2. Запустите игру:

```bash
python game.py
```

## 🎮 Управление

| Действие | Клавиша/Мышь |
|----------|--------------|
| Всосать шарик с поля | **ЛКМ** на шарик |
| Перетащить из инвентаря | **ЛКМ** на шарик в инвентаре + перетащить |
| Выплюнуть шарик | **ПКМ** (правая кнопка мыши) |
| Удалить шарик | Перетащить в красную зону |
| Добавить новый шарик | **N** |
| Сбросить игру | **R** |
| Очистить всё | **C** |
| Пауза | **SPACE** |
| Выход | **ESC** |

## ⚙️ Настройка

В начале файла `game.py` можно изменить настройки:

```python
# Размеры окна
WINDOW_WIDTH = 1200
WINDOW_HEIGHT = 800

# Стартовое количество шариков
INITIAL_BALLS_COUNT = 30

# Радиус всасывания
SUCK_RADIUS = 60

# FPS
FPS = 60
```

## 🎯 Как играть

1. **Всасывание шариков**: Наведите курсор на шарик на поле и нажмите **ЛКМ**. Шарик попадёт в инвентарь (показан слева внизу).

2. **Перетаскивание из инвентаря**: Нажмите **ЛКМ** на шарик в инвентаре и перетащите его мышью на игровое поле или в зону удаления. При наведении на шарик в инвентаре появится подсветка и иконка руки 👆.

3. **Выплёвывание шариков**: Нажмите **ПКМ** в любом месте экрана, чтобы быстро выпустить шарик из инвентаря. Шарик полетит в направлении от центра к курсору.

4. **Смешивание цветов**: Когда два шарика касаются друг друга, их цвета смешиваются. Попробуйте создать новые интересные цвета!

5. **Удаление**: Перетащите шарик из инвентаря в красную зону в правом верхнем углу. При наведении на зону удаления появится предупреждение "Удалить!".

## 🏗️ Архитектура

### logic.py
Содержит всю игровую логику:
- `Ball` - класс шарика с координатами, скоростью, радиусом и цветом
- `GameLogic` - основной класс с логикой игры
- Функции смешивания цветов (RGB ↔ HSV конвертация)
- Обработка коллизий и физика

### game.py
Графический интерфейс:
- Визуализация шариков с 3D эффектами (градиенты, блики, тени)
- Перетаскивание шариков drag-and-drop
- Анимации всасывания и перетаскивания
- UI элементы (счётчики, инвентарь, подсказки)
- Обработка ввода мыши и клавиатуры

## 🎨 Особенности визуала

- **3D шарики** с градиентами, несколькими бликами и тенями для реалистичности
- **Динамические тени** при перетаскивании шариков
- **Полупрозрачный круг** вокруг курсора показывает радиус всасывания
- **Анимация всасывания** при захвате шарика
- **Интерактивный инвентарь** с подсветкой при наведении и иконкой руки
- **Визуальная обратная связь** - курсор меняется при перетаскивании
- **Предупреждение при удалении** - текст "Удалить!" над зоной удаления
- **Красивая зона удаления** с иконкой корзины
